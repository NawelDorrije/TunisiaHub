# Squad Assignments

1. **Users Squad** – Owns `users` backend package and `features/users` UI.
2. **Lieux Squad** – Owns `lieux` backend package and `features/lieux` UI.
3. **Events Squad** – Owns `events` backend package and `features/events` UI.
4. **Restaurants Squad** – Owns `restaurants` backend package and `features/restaurants` UI.
5. **Outdoor Squad** – Handles both `campings` and `hebergements` modules across backend/frontend.
6. **Mobility Squad** – Responsible for `covoiturage` backend/frontend and integration touchpoints.
7. **Integration Squad** – Owns `admin`, shared modules, API contracts, and CI/CD governance.

> Each squad can add more detailed README files inside their module folders to track upcoming work.
